# 🏏 IPL Player Intelligence Platform

An AI-powered analytics platform that lets you ask questions about IPL players 
in plain English and get data-driven insights instantly.

🚀 **Live App:** https://ishagarg-ipl-player-intelligence.streamlit.app

## Features
- 🤖 Natural language Q&A about IPL player performance
- 📊 Interactive dashboards with career stats
- 🏆 IPL Player ROI Index — data-driven auction value scoring

## Tech Stack
Python · Supabase (PostgreSQL) · OpenAI API · Streamlit · Plotly

## Run Locally
pip install -r requirements.txt
streamlit run app/main.py
